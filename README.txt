PROJECT DESCRIPTION

The Simple Payroll Management System is a C++ console application that
calculates employee salaries. It records employee information,
calculates gross salary and net salary, determines payment status,
displays payroll details, and saves the results to payroll_report.txt.

FEATURES

-   Enter multiple employee records
-   Record employee ID, name and department
-   Enter basic salary, allowance and deduction
-   Validate payroll inputs
-   Calculate gross salary
-   Calculate net salary
-   Determine payment status
-   Display payroll report
-   Save payroll report to payroll_report.txt

FORMULAS

Gross Salary = Basic Salary + Allowance Net Salary = Gross Salary -
Deduction

PAYMENT STATUS

If Deduction > Gross Salary: Review Required Otherwise: Ready for
Payment

FILES INCLUDED

main.cpp README.txt algorithm.txt pseudocode.txt flowchart.png (or
flowchart.pdf) payroll_report.txt

HOW TO RUN

1.  Compile main.cpp.
2.  Run the program.
3.  Enter employee details.
4.  View payroll_report.txt after execution.
